Wenn Sie mehrere Ansichten definiert haben, können Sie hier wählen, welche auf der Hudson-Startseite angezeigt werden soll. Dies ist nützlich für größere Hudson-Installationen, bei denen die 'All'-Ansicht in der Praxis zu lang ist. Hinweis: Wenn Sie eine andere Ansicht als "All" als Standardansicht auswählen, können Sie die Ansicht 'All' löschen (sofern Sie dies wünschen).
