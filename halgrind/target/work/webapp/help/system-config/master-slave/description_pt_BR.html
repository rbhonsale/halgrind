Descrição opcional deste slave. Este informação é mostrada na tela de configuração do projeto.

Quando você tiver slaves que são diferentes dos outros, é útil usar este campo para explicar o que é diferente. Por exemplo, entrar "Slave Windows" neste campo permitiria aos donos do projeto escolher sempre construir em uma máquina Windows (por exemplo, se eles precisarem de uma ferramenta de construção específica do Windows.)
