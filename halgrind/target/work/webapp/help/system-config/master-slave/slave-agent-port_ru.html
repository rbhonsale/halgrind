Hudson использует порт TCP для общения с агентами подчиненных узлов запущенных через JNLP. Обычно этот порт выбирается автоматически для предотвращения коллизий, но это может сделать управление безопасностью на сетевом уровне более сложным. Если вы не используете JNLP для подчиненных узлов, рекомендуется заблокировать этот порт. Также вы можете указать статически номер порта и сконфигурировать ваш брандмауэр соответствующим образом.
