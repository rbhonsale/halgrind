Указывает пути в веб-просмотрщик CVS репозиториев. Установка этих параметров позволит Hudson генерировать ссылки из списков изменений на страницы просмотрщика. Необходимо указать две записи для каждого CVSROOT.

URL
URL страницы, которая показывает детали об указанном файле в репозитории.
Diff URL
URL страницы, которая показывает детали различия между двумя версиями (diff).

Вы также можете использовать следующие макроподстановки:
%%P Путь к файлу в CVS, например "foo/bar/Zot.java"
%%r Номер ревизии предыдущей версии (для diff URLов)
%%r Номер ревизии новой версии (для diff URLов)
